## これはなに

- PC とかスマホから家電を操作したいオタク向けの統合スマートホームマネージャー
- Nature Remo と Kasa のスマートプラグに対応
- Discord エージェントからの自動制御に対応 (管理画面は WIP)
- WebHook API で IFTTT 等からの細かな連携が可能 (管理画面は WIP)
